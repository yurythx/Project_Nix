Informações Importantes
Backup de Banco de Dados
  • Substitui todos os dados do banco
  • Usuários e sessões podem ser afetados
  • Recomenda-se fazer logout de todos os usuários
Backup de Mídia
  • Substitui arquivos de mídia existentes
  • Arquivos não presentes no backup serão mantidos
  • Processo pode demorar dependendo do tamanho
Backup de Configuração
  • Restaura configurações do sistema
  • Pode requerer reinicialização
  • Configurações personalizadas podem ser perdidas
